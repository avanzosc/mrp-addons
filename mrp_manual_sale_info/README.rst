====================
MRP Manual Sale Info
====================

This module extends the functionality of Odoo for adding manually a sale order and it's
related fields to Manufacturing Orders and Work Orders:

  * Sale Order
  * Customer
  * Commitment Date
  * Customer Reference

**Table of contents**

.. contents::
   :local:

Usage
=====

Select sale order in manufacturing orders and new fields are displayed in tree and
in form views.
