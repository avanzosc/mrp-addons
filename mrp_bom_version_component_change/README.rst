================================
Mrp bom version component change
================================

* This module allows to change massively one component by another on a list of
  BoMs. If you checked the new fied "Create new BoM version", before you
  compose a massive change in BoMs, the BoM is to be changed It will state
  "Historical", and changing component is made in a new version of the BoM.
