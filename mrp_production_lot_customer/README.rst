===========================
MRP production lot customer
===========================

* In manufacturing orders show the customer of th production lot.
* When checking availability, search for lots of components that are from the
  same customer or have an empty customer.

Bug Tracker
===========

Bugs are tracked on `GitHub Issues
<https://github.com/avanzosc/mrp-addons/issues>`_. In case of trouble,
please check there if your issue has already been reported. If you spotted
it first, help us smash it by providing detailed and welcomed feedback.

Do not contact contributors directly about support or help with technical issues.
