==============================
MRP Work Order Product Variant
==============================

* Product variant attributes in operation lines of the BoM.
* In a production order, in the BoM onchange, a work order operation only 
  will be created if the operation line of the BoM has no variant attribute or
  if the product to be produced also has this attribute.
